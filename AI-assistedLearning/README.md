# AI-assisted Programming Learning Assistant and Code Reviewer

This tool empowers you to leverage AI as a learning assistant and code reviewer, not as a shortcut to answers. Use it to:
- Review and understand code patterns
- Get guidance on debugging and problem-solving
- Learn best practices and improve code quality
- Receive constructive feedback on your implementations

Remember: The goal is to deepen your understanding and develop your programming skills, not to bypass the learning process.
